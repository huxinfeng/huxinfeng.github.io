## 👋 欢迎！

博客站点使用以下技术创建

- 使用 [Astro](https://astro.build) 框架开发，具有非常友好的 SEO 能力
- 前端使用 github page 部署，后端使用 github discussions 作为数据库存储
- 引入 [cloudflare](https://www.cloudflare.com)，使用无计量的 DDoS 保护、全球 CDN 及其他产品，保护和加速网站
- 评论系统使用 [giscus](https://giscus.app) 桥接，这样游客即可在帖子下面浏览所有评论；并且可以登录 github，发表自己的评论
- 使用 github actions 能力，无论是新增、修改、删除 discussions，都会自动触发 actions 构建，自动化部署
